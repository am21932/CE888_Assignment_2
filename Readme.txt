# Getting the datasets
	1. The three datasets named CrimeClassification.csv, Forex_chfjpy and Skin_nonskin.txt  are available inside the zipped file named Data.zip.
	2. The four code files required are Skin_NonSkinClassification.ipynb, CrimeIncidents.ipynb, ForexClassification.ipynb and Imbalanced_dataset_utility.py.

#Running the code
	To run the code please place the three datasets and the ipynb and the py files in the same directory.
